# Installation & Setup

## Prerequisites

Before installing the plugin, ensure the target environment meets the minimum standards:
- **WordPress**: 6.4+
- **PHP**: 7.4+

## Installation Instructions

1. Clone or download the repository to your local machine.
2. Navigate to your WordPress installation's `wp-content/plugins/` directory.
3. Upload/Copy the `plugin` folder from this repository into `wp-content/plugins/` and rename it to `wcasia-event-submissions`.
4. Log into your WordPress Dashboard.
5. Go to **Plugins** -> **Installed Plugins**.
6. Find "WordCamp Asia Event Submissions" in the list and click **Activate**.

You are now ready to start using the plugin by embedding the shortcode on your desired page.
